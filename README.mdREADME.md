# Lumière Privée — Luxury Concierge Landing Page

A modern, elegant landing page built for a luxury concierge service based in Casablanca.

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Modern UI with React & Tailwind CSS
- Dark / Light mode toggle
- Multi-language support (EN / FR)
- Smooth animations (Framer Motion)
- Clean and structured components

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion
